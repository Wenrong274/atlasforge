# AtlasForge v0.1.2

## 新增

- 新增 Atlas 預覽選取高亮：點選左側幀列表時，右側 Atlas 預覽會標示對應子圖位置。
- 強化序列拆圖預覽操作：支援 Ctrl + 滾輪縮放與拖曳平移。
- 匯出 JSON 的 `meta.version` 會使用實際應用程式版本。

## 修正

- 修正 plist 拆圖解析：支援更穩定的 key/dict 配對、浮點座標與旋轉幀尺寸還原。
- 修正網格拆圖在欄列為 0 時可能除以零的問題。
- 修正 Atlas/Frame 預覽與重新打包流程中的 SkiaSharp bitmap 釋放，降低記憶體累積。

## 測試與維護

- 將拆圖 roundtrip 測試改為產生臨時測試圖片，不再依賴本機絕對路徑。
- 移除本機檔案依賴的 no-op 測試，保留可重跑的 DPI 行為測試。
- 發布流程會優先使用對應 tag 的 release note 檔案。
